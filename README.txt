MVP for Fire Predictor

Application will only run on a device that can run windows executables

Instructions to run:
Unzip file
Open mvp.exe (either double click on file or open through command prompt)
	preferable to use cmd to open since project runs through it
No other libraries/dependencies needed

Usage:
Wait for at most 3 minutes for everything to be loaded
The first thing it says should be for user to input max temperature
Once all environmental factors have been inputted, model will output graph with prediction and breakdown of feature importances
